1. 核心理念：纯 RNN + Token/Channel Mixing

# RWKV-4 核心: WKV operator
# 加权 Key-Value (带指数衰减)

def wkv_operator(w, u, k, v):
    """
    w: channel-wise decay (可学习, 每通道不同)
    u: bonus for current token
    k, v: key, value sequences [T, C]
    """
    T, C = k.shape
    outputs = []
    state_n = torch.zeros(C)  # 分子累加
    state_d = torch.zeros(C)  # 分母累加
    for t in range(T):
        # 当前 token bonus
        wkv_t = (state_n + torch.exp(u + k[t]) * v[t]) \
               / (state_d + torch.exp(u + k[t]))
        outputs.append(wkv_t)

        # 状态更新: 指数衰减历史
        state_n = torch.exp(-torch.exp(w)) * state_n \
                  + torch.exp(k[t]) * v[t]
        state_d = torch.exp(-torch.exp(w)) * state_d \
                  + torch.exp(k[t])
    return torch.stack(outputs)

# RWKV Block = Token Mixing + Channel Mixing
def rwkv_block(x):
    # Token Mixing (时间方向交互)
    x = x + time_mixing(layer_norm(x))
    # Channel Mixing (通道方向 FFN)
    x = x + channel_mixing(layer_norm(x))
    return x

RWKV 的独特设计

RWKV 不用任何 attention matrix，而是用 WKV operator — 一种带指数衰减的加权聚合：

$\text{wkv}_t = \frac{\sum_{i=1}^{t-1} e^{-(t-1-i)w + k_i} v_i + e^{u+k_t} v_t}{\sum_{i=1}^{t-1} e^{-(t-1-i)w + k_i} + e^{u+k_t}}$

关键组件：

符号含义
w通道级衰减率 (可学习)
u当前 token bonus
RReceptance (接收门控)
Time Shift：RWKV 的一个简单但有效的创新 — 将 x_t 和 x_{t-1} 做线性插值后再投影，让每个 token "看到"前一步信息。

2. RWKV-5/6 演进

# RWKV-5 Eagle: multi-head WKV
# 关键改进: data-dependent decay

def rwkv5_wkv(r, k, v, w, u):
    """RWKV-5: multi-head, 矩阵值状态"""
    B, H, T, D = k.shape  # multi-head!
    state = torch.zeros(B, H, D, D)
    outputs = []
    for t in range(T):
        # w_t 现在是 data-dependent!
        # (RWKV-4 的 w 是固定的)
        kv = k[:,:,t:t+1,:].transpose(-1,-2) \
             @ v[:,:,t:t+1,:]
        state = state * w[:,:,t].unsqueeze(-1) + kv
        out = r[:,:,t:t+1,:] @ state
        outputs.append(out)
    return torch.cat(outputs, dim=2)

# RWKV-6 Finch: LoRA-style delta
# 在 time_shift 中引入低秩更新

def rwkv6_time_mixing(x, state):
    # LoRA-style 数据依赖插值
    # λ = sigmoid(x @ A @ B)
    # shifted = λ * x_t + (1-λ) * x_{t-1}
    # → 更精细的 token 间信息融合
    lora_out = x @ A @ B  # 低秩
    lam = torch.sigmoid(lora_out)
    shifted = lam * x + (1 - lam) * state
    return shifted

RWKV-6 进一步引入 LoRA-style 的低秩参数化，让 time shift 中的插值系数也变为数据依赖。

3. 训练与推理

# 训练模式: 并行 WKV kernel (Triton/CUDA)
# 将递推展开为并行 scan

# 推理模式: 纯 RNN, O(1) per token
class RWKVInference:
    def __init__(self, model):
        self.model = model
        # 状态: 每层 2 个值
        # time_mixing state + channel_mixing state
        self.states = [
            {'time': zeros(H, D, D),
             'shift': zeros(D)}
            for _ in range(n_layers)
        ]

    def forward_one_token(self, token_id):
        x = self.model.embed(token_id)
        for i, layer in enumerate(self.model.layers):
            # O(D²) per layer — 恒定!
            x, self.states[i] = layer.forward_rnn(
                x, self.states[i]
            )
        logits = self.model.head(x)
        return logits

# 推理:
#   恒定 O(n_layers * D²) 状态

双模式一致性

RWKV 的核心优势：训练和推理使用完全相同的数学，只是实现策略不同：

$\text{训练: parallel scan (一次性处理 N tokens)}$ $\text{推理: sequential RNN (逐 token O(1))}$ $\text{两者数学等价，结果完全一致}$
