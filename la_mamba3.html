1. 三大创新 — SSM 独有的视角

# Mamba-3 的三个核心改进:
# 每个都来自 SSM 独有的理论视角
# (Linear Attention 视角无法自然推导出这些)

# === 1. 梯形离散化 (Exponential-Trapezoidal) ===
# Mamba-1/2: Euler 法 (一阶, O(Δ²) 误差)
#   h_t = exp(Δ·A)·h_{t-1} + Δ·B·u_t
#
# Mamba-3: 梯形法 (二阶, O(Δ³) 误差)
#   h_t = α_t·h_{t-1} + β_t·B_{t-1}·x_{t-1} + γ_t·B_t·x_t
#   其中 λ_t ∈ [0,1] 数据依赖
#
# 效果: 等价于状态内的宽度-2卷积
# → 可以移除外部 short convolution!

# === 2. 复数态 (Complex-Valued State) ===
# Mamba-2: A_t ∈ R (只能单调衰减)
# Mamba-3: A_t = A_real + i·θ_t (可以旋转!)
#
# 离散化后 = 带数据依赖 RoPE 的实值 SSM
# → 实现: 对 B,C 应用 data-dependent RoPE
# → 效果: 100% Parity (Mamba-2: 0.9%)

# === 3. MIMO (多输入多输出) ===
# SISO: B ∈ R^N, x ∈ R^P (单输入单输出)
# MIMO: B ∈ R^(N×R), x ∈ R^(P×R), rank R=4
# → 外积变矩阵乘, 利用 tensor cores
# → 4x FLOPs, 但延迟几乎不增 (compute bound!)
# → 算术强度: ~2.5 → ~10 ops/byte

SSM 视角的独特价值

这三个改进都源自 SSM/控制论的理论，不是从 Linear Attention 视角能自然推导的：

1. 离散化改进 — LA 没有"连续到离散"的概念，无法讨论"更高阶积分器"。

2. 复数态 — LA 中 A 代表"遗忘率"，复数没有意义；但 SSM 中 A 是状态转移矩阵，复数特征值 = 振荡模式。

3. MIMO — 经典控制论概念（多输入多输出系统），LA 中没有对应物。

核心洞察：Mamba-3 证明了 SSM 视角的不可替代性。虽然 Mamba-2 证明了 SSM = LA，但不同视角带来不同的设计灵感。Mamba-3 的创新不可能从 Linear Attention 或 TTT 视角产生。

推理优先 (Inference-First)

Mamba-3 的设计哲学是"推理效率第一"。MIMO 的精妙之处：增加 4x FLOPs 但几乎不增延迟，因为 decode 是 memory-bound 的（算术强度只有 2.5 ops/byte，H100 峰值 295）。

MIMO 把外积换成矩阵乘，利用了闲置的计算单元（tensor cores），把 ops/byte 从 2.5 提升到 10 — 仍远低于硬件峰值，所以延迟几乎不变。

核心结论：Mamba-3 是第一个在 1.5B scale 全面超越 GDN 的纯 recurrent 模型 (+1.8 pts)。加上最快的推理速度，它重新定义了 efficiency-quality Pareto 前沿。

3. Mamba 家族演进

# Mamba 家族演进:
#
# Mamba-1 / S6 (2023.12):
# - 选择性 SSM (data-dependent B, C, Delta)
# - 对角 A 矩阵
# - Hardware-aware parallel scan
# - 首次在 LM 上匹配 Transformer
#
# Mamba-2 / SSD (2024.05):
# - A 简化为 scalar*I → 可用 matmul
# - State Space Duality 定理:
#   SSM = 带衰减的 Linear Attention
# - 8x 训练加速 (vs Mamba-1)
# - 牺牲表达力换效率
#
# Mamba-3 (2026.03):
# - 恢复表达力 (复数态+梯形)
# - 保持 matmul 效率 (SSD 框架内)
# - MIMO 提升 decode 利用率
# - 移除 short convolution
# - 推理比 Mamba-2 还快!
#
# 趋势: 表达力↑ + 效率↑ (不再 trade-off)
# 方法: 从 SSM 理论中挖掘免费的改进

# 与 Gated DeltaNet 的关系:
# GDN: LA 视角的最强 → delta rule + gate
# Mamba-3: SSM 视角的最强 → 复数态 + MIMO
# 两者互补, 可以混合 (Super Apriel)

SSM vs Linear Attention: 竞争还是互补？

Mamba-3 和 GDN 分别代表 SSM 视角和 LA 视角的最强模型。Mamba-3 在 accuracy 上超越 GDN (+1.8)，但 GDN 有更丰富的记忆管理机制 (erase/write)。

实践中两者互补 — Super Apriel 等混合架构已经开始每层选择不同的 mixer。

核心启示：Mamba-3 证明了不同理论视角（SSM vs LA vs TTT）不是竞争关系，而是设计灵感的来源。最好的模型将综合所有视角的最佳想法。
