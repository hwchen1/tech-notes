1. 核心理念：Memory as Neural Network

# Titans: 记忆不是矩阵 S，而是一个神经网络 M
# M 用自己的梯度下降来"学习"输入序列

class TitanMemory(nn.Module):
    def __init__(self, d_model, mem_dim):
        self.M = nn.Linear(d_model, mem_dim)
        # M 的参数就是"记忆"
        # 通过 surprise-based 更新学习

    def forward(self, x_t, y_t):
        # 1. 用记忆预测
        y_pred = self.M(x_t)

        # 2. 计算 surprise (预测误差)
        surprise = (y_pred - y_t).pow(2).sum()

        # 3. 通过梯度更新记忆参数
        # (类似 meta-learning 的 inner loop)
        grad = torch.autograd.grad(surprise, self.M.parameters())
        with torch.no_grad():
            for p, g in zip(self.M.parameters(), grad):
                # 遗忘 + 学习
                p.data = theta * p.data - eta * g

        # 4. 读取记忆
        return self.M(x_t)

# 关键: 只记住让模型"惊讶"的信息
# surprise 低 → 已知信息, 少更新
# surprise 高 → 新信息, 多更新

超越简单累加

传统 linear attention 的根本问题：对所有 token 等权重写入状态，重要信息和噪声都一样处理。

Titans 的核心洞察：用预测误差 (surprise)决定记忆更新强度：

$\text{surprise}_t = \|M(x_t) - y_t\|^2$ $\theta_t = \theta_{t-1} \cdot \text{decay} - \eta \cdot \nabla_\theta \text{surprise}_t$

这等价于一个 online meta-learning 过程：记忆网络 M 在处理序列时不断"学习"，但只对让它惊讶的信息做大的权重更新。

与 Linear Attention 的联系：当 M 是单层线性网络时，Titans 退化为带 surprise-weighted 的 DeltaNet。Titans 的创新在于允许 M 是多层非线性网络。

2. 三种记忆集成方式

# Titans 提出三种集成变体:

# 1. MAC: Memory as Context
def mac_layer(x, memory):
    # 记忆输出拼接为额外 context
    mem_out = memory.read(x)   # [B, d]
    ctx = torch.cat([mem_out.unsqueeze(1), x], dim=1)
    out = attention(ctx)       # 标准 attention
    memory.update(x)
    return out[:, 1:]  # 去掉 mem token

# 2. MAG: Memory as Gate
def mag_layer(x, memory):
    # 记忆输出门控融合
    mem_out = memory.read(x)
    attn_out = attention(x)
    gate = sigmoid(linear(x))
    out = gate * mem_out + (1 - gate) * attn_out
    memory.update(x)
    return out

# 3. MAL: Memory as Layer
def mal_layer(x, memory):
    # 记忆作为独立的一层
    # (替换某些 attention 层)
    mem_out = memory.read(x)
    memory.update(x)
    return mem_out  # 纯记忆输出

推荐: MAC (Memory as Context)

三种方式各有适用场景：

变体优势适用
MAC质量最强长上下文 QA
MAG效率-质量平衡通用语言模型
MAL最轻量端侧部署

MAC 将记忆读取结果作为额外的 context token 拼接到 attention 输入中，让 attention 机制自己学会如何利用长期记忆。

关键突破：在 BABILong 1M-token recall 任务上，MAC 超越 GPT-4、Qwen2.5-72B、Llama3.1-70B 等超大模型（用远小的参数量），证明 surprise-based memory 在超长上下文检索上具有碾压性优势。

3. 与传统 Linear Attention 的本质区别 + 实验结果

# 对比: 传统 LA 的状态更新
# S_t = α·S_{t-1} + k_t⊗v_t
# → Additive (加法规则)
# → 无法学习非线性关联

# Titans 的记忆更新
# θ_t = θ_{t-1}·decay - η·∇_θ L(x_t, y_t)
# → Gradient-based (梯度规则)
# → 可以学习任意非线性映射

# 特殊情况: 单层线性 Memory
# M(x) = W·x, 则:
# L = ||W·k - v||²
# ∇W = 2·(W·k - v)·k^T
# W_new = decay·W - η·(W·k - v)·k^T
#       = decay·W - η·W·kk^T + η·v·k^T
# 这就是 DeltaNet! (with decay = GatedDeltaNet)

# 所以: Gated DeltaNet ⊂ Titans
# Titans 允许 M 是多层非线性网络
# → 记忆容量和表达力远超 d×d 矩阵
