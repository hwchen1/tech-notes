1. 高分辨率 ViT 设计

# SAM ViT-H 配置
image_size    = 1024      # 极高分辨率!
patch_size    = 16

hidden_dim    = 1280

mlp_ratio     = 4
window_size   = 14        # 14×14 patches

# Attention 模式 (32 层):
# 大部分层: Window Attention
# 每隔几层: Global Attention
# pattern: [W,W,W,G, W,W,W,G, ...]

# Window attention:
# 将 64×64 grid 分为 14×14 窗口
# 每窗口内独立 self-attention
# → O(window² × d) 而非 O(N² × d)

# Global attention (每4层一次):
# 所有 4096 tokens 全局 attend

Window + Global

1024² 输入产生 4096 tokens，全局 attention 计算量过大。解决方案：大部分层用 window attention，少数层用 global attention。

$\text{Window: } O(w^2 \cdot d) \times N_{\text{windows}}$ $w = 14 \times 14 = 196, \quad N_w \approx 21$ $\text{Global: } O(4096^2 \cdot d) \text{ (每4层一次)}$

高分辨率的必要性

分割任务需要精细的边界信息。1024² 输入 + 16×16 patch → 64×64 特征图，足以描述细粒度边界。

2. SAM 完整架构

SAM 的设计思想：重的 image encoder 只编码一次，轻的 decoder 可以对同一张图反复用不同 prompt 推理
