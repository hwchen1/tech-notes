1. 自蒸馏 + Masked Modeling

# DINOv2 训练: DINO + iBOT 结合

# Student 和 Teacher 共享架构
student = ViT_g()
teacher = EMA(student)  # 指数滑动平均

for images in dataloader:
    # 1. DINO loss (全局自蒸馏)
    # student: local crops → global target
    s_cls = student(local_crops).cls
    t_cls = teacher(global_crops).cls
    loss_dino = CE(s_cls / τ_s, t_cls / τ_t)

    # 2. iBOT loss (patch 级 MIM)
    # 遮挡 student patches, 预测 teacher
    s_patch = student(masked_img).patches
    t_patch = teacher(full_img).patches
    loss_ibot = CE(s_patch[mask], t_patch[mask])

    # 3. 总 loss
    loss = loss_dino + loss_ibot

    # 4. 更新 teacher (EMA)
    teacher = momentum * teacher + (1-m) * student

双重目标

DINO: [CLS] token 自蒸馏 → 学全局语义（物体识别、场景分类）

iBOT: Masked patch 预测 → 学局部细节（边缘、纹理、空间关系）

$\mathcal{L}_{\text{DINO}} = -\sum p_t \log p_s, \quad p = \text{softmax}(z/\tau)$ $\mathcal{L}_{\text{iBOT}} = -\sum_{i \in \text{mask}} p_t^{(i)} \log p_s^{(i)}$ $\text{Teacher: } \theta_t \leftarrow m \cdot \theta_t + (1-m) \cdot \theta_s$
