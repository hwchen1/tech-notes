MAE

Meta 2022 — Masked Autoencoders: 75% 随机遮挡 + 像素重建，极简高效的自监督 ViT 预训练

1. 高比例随机遮挡

# MAE 训练流程

# 1. 将图像分为 patches
patches = patchify(image)  # 196 patches

# 2. 随机遮挡 75%
mask = random_mask(ratio=0.75)
visible = patches[~mask]   # 仅 49 patches!

# 3. 仅对可见 patch 编码
encoded = encoder(visible)  # 超高效
# encoder 只处理 25% tokens
# → 训练速度 3-4× faster

# 4. 补回 mask tokens 后解码
full = insert_mask_tokens(encoded, mask)
decoded = decoder(full)

# 5. 只在被遮挡位置计算 loss
loss = MSE(decoded[mask], patches[mask])

核心洞察

图像信息高度冗余 — 遮挡 75% 后仍可重建。这迫使模型学习语义而非简单插值。

encoder 只处理可见的 25% patches → 计算量降低 4×，训练极其高效。

$\text{mask ratio} = 75\%$ $\text{encoder input: } 196 \times 25\% = 49 \text{ tokens}$ $\mathcal{L} = \text{MSE}(\hat{x}_{\text{mask}}, x_{\text{mask}})$

vs NLP (BERT)

BERT 只遮 15%，因为语言信息密度高。图像冗余度远高于文本，所以 MAE 可以遮 75%。

2. 非对称 Encoder-Decoder

非对称设计：大 encoder 学表征 + 小 decoder 做重建。训练后只保留 encoder。decoder 只占 ~10% FLOPs

$\text{Encoder: ViT-L (304M)} \quad \text{Decoder: 8层 (27M) — 仅占 8\% 参数}$ $\text{重建目标: 归一化后的 patch 像素值 (per-patch normalize)}$ $\hat{x}_p = \frac{x_p - \mu_p}{\sigma_p}, \quad \mathcal{L} = \|\text{dec}(z) - \hat{x}_{\text{mask}}\|_2^2$
