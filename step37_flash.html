参数量不变 (都是 196B)，但 Agent 能力暴涨：

$\text{GDPval: } +64\%$ $\text{ClawEval: } +54\%$ $\text{Toolathlon: } +49\%$ $\text{HLE: } +32\%$ $\text{SWE-Pro: } +10\%$

推测提升来源：

1. 大量 Agent/Tool-use 数据

2. RL from Agent feedback

3. 长程任务的对齐优化

4. 搜索和工具调用的稳定性训练

不是架构变化（同 196B），而是 post-training 阶段的大幅投入。
