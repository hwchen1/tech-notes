1. 从 AGI 到 ASI 的四条路径

// 核心问题:
// 假设 human-level AGI 实现后,
// AI 进步如何继续?

// 定义:
// AGI = median human-level intelligence
// ASI = 远超人类组织的 general intelligence

// Pathway 1: Scaling
// 更大模型 + 更多数据 + 更多计算
// Effective compute ~4×/yr (过去十年)
//   = Hardware 1.5×/yr
//   × Investment 2.5×/yr
// Algorithmic efficiency ~3×/yr
// → 合计可能 10×/yr 有效计算增长

// Pathway 2: Paradigm Shift
// 全新架构/优化方法
// 例: neuromorphic, RL pretrain,
//     explicit world models
// 本质上不可预测

// Pathway 3: Recursive Self-Improvement
// AI → better AI → even better AI → ...
// 可能爆炸性增长 (超指数)
// 也可能快速饱和
// 无历史先例

// Pathway 4: Multi-Agent Collective
// 多 AGI 协调 → Group Agent
// 类似 "自动化公司"
// 涌现行为难预测

// 四条路径不互斥, 可并行发展

核心框架

AGI: 大致等于 median human-level intelligence 的系统。ASI: 远超人类水平的 general intelligence — 比"大型人类组织"更聪明的系统。

四条路径

Scaling: 最直接 — 更多 compute → 更大模型 → 更多 test-time search。有效计算过去十年 ~4×/yr，预计可延续到下一个十年。

Paradigm Shift: 全新架构 (spiking neurons, neuromorphic, analog computing) 或全新训练范式 (RL pretrain, explicit world models)。作为 scaling 天花板后的替代方案。

Recursive Self-Improvement: AI 改进自身的四种方式 — (1) 更好代码/架构, (2) 更好硬件, (3) 更好数据 (AlphaZero 式), (4) 分工协作。可能超指数增长。

Multi-Agent: 多个 AGI 协调形成 Group Agent。高带宽通信 + 无限复制 = 人类组织无法比拟的集体智能。

关键观点: 四条路径不互斥，可并行。即使单模型进步停滞，Multi-Agent scaling 仍可推动 ASI。

2. 六大摩擦与瓶颈 (Frictions)

// Friction 1: Data Wall
// 高质量文本数据预计本十年内耗尽
// 缓解: 合成数据, self-play, RL sim
// AlphaZero 式: search → distill → repeat

// Friction 2: Economic/Resource Growth
// 能源、芯片、投资需指数增长
// 内存带宽 + 互连瓶颈
// 缓解: AI 提升经济回报 + 效率

// Friction 3: Neural Paradigm 不够
// Transformer+SGD+Scaling 可能有天花板
// 缓解: paradigm shift 研究

// Friction 4: Research Gets Harder
// "Ideas get harder to find" (Bloom 2020)
// Moore's Law: 今天需要 18× 研究员
//   (vs 1970s) 来维持相同进度
// 缓解: AI 研究员可快速复制扩容

// Friction 5: Abstraction Barrier ⚠️最深刻
// 模型训练在人类抽象上
// → 无法发现全新概念 (force, causality)
// → 可能卡在 AGI 水平
// 除非: grounded concept discovery
//   从 raw sensor data 抽象新概念

// Friction 6: Complexity-Theoretic Limits
// 某些问题可能计算不可解
// AIXI 是不可计算的上界
// 有限资源下必须近似

Abstraction Barrier (最深刻)

核心论点: 当前 LLM 训练在人类生成的抽象上。如果给模型喂入同量 token 但内容限制在牛顿之前的科学知识 — 它不可能推导出广义相对论或量子力学。

缺失能力: 从 raw sensor data 发现全新概念 (force, causality)。人类社会花了数千年才达到当前的世界理解水平。

影响: 如果这是真的，单个 AI 可能卡在 AGI 水平。但集体 ASI 仍可通过 Multi-Agent scaling 实现。

Data Wall

现状: 高质量文本数据预计本十年内耗尽。

AlphaZero 式缓解: Base model → test-time search → 改进结果 distill 回 base model → 循环。只要有 verifier (像国际象棋的胜负判定)，就能自我提升。

Research Gets Harder

Bloom et al. 2020: 维持 Moore's Law 今天需要 18× 研究员 (vs 1970s)。Ideas get harder to find。

AI 的优势: 人工研究员可瞬间复制 18×。训练人类需要年，扩容 AI 只需 compute。

论文结论: "不能轻易否认在未来十到二十年内从 AGI 直接进入 ASI 的可能性"。若多个摩擦非硬阻碍，过渡可能相对平滑。

3. 递归自我改进 & 研究议程

// 递归自我改进的四种形式:

// (1) Code/Algorithm Improvement
// AI 写更好架构/优化器/搜索算法
// → next-gen AI systems
// 已有: NAS, auto-HPO

// (2) Hardware Improvement
// AI 设计更好芯片 + 制造流程
// 速度受限于物理制造
// 已有: Mirhoseini 2021 (Google)

// (3) Data Improvement (最微妙)
// AI 生成/筛选更好训练数据
// AlphaZero 式: policy+value net
//   → guide search → distill back
// → 递归提升 (proven in games)

// (4) Division of Labor (sociogenic RSI)
// 专业化 → 效率↑ → 释放资源
// → 更多专业化 → 循环
// 目前 foundation model 都是 generalist
// 专家集体的经验数据很少

// === 研究议程 (7 大方向) ===
// 1. Scaling 瓶颈量化
// 2. 定量预测模型
//    (scaling laws + 经济模型耦合)
// 3. ASI Benchmarking
//    (超越人类参照的评估)
// 4. 递归改进度量与预测
//    ("RSI scaling laws")
// 5. Multi-Agent Scaling Laws
// 6. Universal AI 理论实际化
// 7. Safety & Alignment

递归改进的加速与阻尼

加速因素: 纯数字研究员速度无限制，可并行复制，跨实例共享知识 (权重 merge)，无需年薪/培训/退休。

阻尼因素: 即使超人速度的数字研究员也必须等待实验结果。涉及物理操作 (芯片制造) 无法任意加速。

AlphaZero 类比

关键洞察: Test-time search 改进 base model 输出 → distill 回 base model → 更好的 search prior → 循环。只要有良好的 verifier，递归蒸馏就能持续改进。开放问题: 什么领域有足够好的 verifier?

数字智能的根本优势

论文列出 AI 相对人类随 compute 增长的优势:

速度 (think faster) / 并行 (多实例) / 记忆 (perfect recall) / 通信 (高带宽) / 复制 (克隆) / 可编辑 / 持久 / 可组合

论文立场: "即使假设 AI progress 只靠 scaling 且单模型停滞，仅靠量化增长 (更多实例 + 更快 + 更长 think time) 也可能解锁看似需要质变的能力。"
时间线: "未来十到二十年内从 AGI 进入 ASI 不能被轻易否定。" 但预测高度不确定 — 可能性范围从 "本十年末" 到 "几十年"。
