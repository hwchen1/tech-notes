1. 核心问题 — Titans 的三个局限

# Titans 的三个局限:
# 1. 有限容量: 固定大小 memory 无法随序列增长
# 2. 在线更新: 只能向前看，不能回顾优化
# 3. 记忆管理: 无法主动选择记忆/遗忘什么

# ATLAS 的三个解法:
# 1. DeepTransformers — 多层记忆网络
# 2. 两阶段更新 — 先全局扫描,再精细优化
# 3. 主动记忆管理 — 学习何时读/写/忘

# DeepTransformers 架构:
# 标准 Transformer: 每层独立计算
# DeepTransformer: 层间共享压缩记忆
#   Memory_l = f(Memory_{l-1}, context_l)
# → 记忆容量随深度增长!

# 核心公式:
# 记忆更新 (surprise-based, 同 Titans):
# θ -= η · ∇L(θ; x_t)  if surprise > threshold
#
# ATLAS 新增:
# 两阶段:
#   Stage 1: 粗粒度全局扫描 (大 chunk)
#   Stage 2: 细粒度局部优化 (小 chunk)
# → 类比人类: 先浏览全文,再精读重点

从 2M 到 10M Context

Titans 在 2M context 上展示了可能性，ATLAS 通过 DeepTransformer 架构将这推到 10M。核心洞察：memory 应该像 Transformer 本身一样是多层的，每层提取不同粒度的信息。

DeepTransformer 的核心思想：传统 Transformer 每层独立计算，层间不共享状态；DeepTransformer 让相邻层共享压缩记忆，使记忆容量随网络深度线性增长。

两阶段更新模拟人类阅读策略：

阶段策略类比
Stage 1大 chunk, 粗粒度浏览全文
Stage 2小 chunk, 细粒度精读重点
ATLAS = Titans + 深层记忆 + 两阶段更新 + 主动管理。在 BABILong 10M context 上达到 80%+ accuracy，是第一个在这种极长上下文上有实质性能力的模型。

2. 统一理论框架 Miras

# Miras 框架 (2504.13173):
# 统一视角 — 所有序列模型都是 associative memory
#
# 四个设计轴:
# 1. Memory Architecture: 矩阵/向量/网络
# 2. Bias Objective: 什么样的 loss 驱动记忆
# 3. Retention Gate: 如何遗忘
# 4. Learning Algorithm: SGD/momentum/Adam...
#
# 实例化:
# Transformer = {矩阵 KV, softmax bias, 无gate, 无学习}
# GDN = {矩阵 S, L2 loss, α gate, SGD}
# Titans = {网络 θ, surprise loss, η gate, gradient}
# ATLAS = {深层网络, multi-obj, adaptive gate, 2-stage}
#
# 新架构 (从 Miras 推导):
# Moneta: memory + attention bias → 最优解析解
# Yaad: multi-resolution memory bank
# Memora: continual learning 式记忆

# TNT 训练加速 (2511.07343):
# Titans/ATLAS 的训练瓶颈: BPTT through memory
# TNT 解法: hierarchical chunking
#   大 chunk → 全局 context (memory 层)
#   小 chunk → 局部 detail (并行计算)
# → 17x speedup, 同时提升 accuracy!

所有模型都是 Associative Memory

Miras 的核心洞察：Transformer 的 attention 本质是一种"无学习"的 memory lookup，而 Titans/ATLAS 的 memory 是"有学习"的 memory write+read。统一后可以系统性地设计新架构。

每个序列模型都可以定位到四个轴的某个组合：Memory Architecture 决定容量上限，Bias Objective 决定写入策略，Retention Gate 决定遗忘方式，Learning Algorithm 决定优化质量。

Titans 的影响力

Titans 在 18 个月内产生了 100+ 引用和 8+ 直系后续论文，形成了一个完整的研究生态。核心贡献不是某个具体模型，而是"memory 应该用梯度学习"这个范式转换。

NVIDIA 证明 TTT (Test-Time Training) ≡ Linear Attention。这意味着 Titans 和 GDN 实际上是同一事物的不同视角——一个从"记忆学习"出发，一个从"线性注意力"出发，殊途同归。

核心观点演进: 2024 — "memory 可以用梯度更新"; 2025 — "所有模型本质都是 memory" + "TTT = Linear Attention"; 2026 — "memory 需要多层+两阶段"。这条线从 Titans 到 ATLAS，构建了一个完整的 gradient-based memory 理论体系。
