Multi-Token Prediction

一次预测多个 token — 更好的表征学习 + 推理加速

1. Next-Token vs Multi-Token

MTP 从每步 1 个预测目标扩展到 k 个，迫使模型学习更长远的规划，同时为推理加速提供 draft tokens

2. 架构设计（DeepSeek 方案）

MTP 架构：共享 backbone 输出 hidden states，多个轻量 prediction head 分别预测 t+1, t+2, ..., t+k

DeepSeek-V3 设计要点：每个 MTP module 接收前一步的 hidden state + 对应位置的 ground-truth embedding（teacher forcing），经过 1 层 Transformer 后用共享的 output head 预测。训练时 k=2 即有明显收益。

3. 训练实现

import torch
import torch.nn as nn

class MTPModule(nn.Module):
    """One MTP prediction head (predicts token at position t+k)"""
    def __init__(self, d_model, n_heads):
        super().__init__()
        self.proj = nn.Linear(d_model * 2, d_model)  # merge h_t + emb(x_{t+k-1})
        self.layer = nn.TransformerEncoderLayer(d_model, n_heads, d_model*4, batch_first=True)
        self.norm = nn.RMSNorm(d_model)

    def forward(self, hidden, prev_emb):
        # hidden: backbone output [B, N, D]
        # prev_emb: embedding of ground-truth token at t+k-1 [B, N, D]
        x = self.proj(torch.cat([hidden, prev_emb], dim=-1))
        x = self.layer(self.norm(x))
        return x


class MTPLoss(nn.Module):
    def __init__(self, backbone, embed, lm_head, n_extra=2, lam=0.5):
        super().__init__()
        self.backbone = backbone
        self.embed = embed        # shared embedding
        self.lm_head = lm_head    # shared output projection
        d = embed.embedding_dim
        self.mtp_modules = nn.ModuleList([MTPModule(d, 8) for _ in range(n_extra)])
        self.n_extra = n_extra
        self.lam = lam

    def forward(self, input_ids, labels):
        h = self.backbone(self.embed(input_ids))  # [B, N, D]

        # Standard next-token loss
        logits = self.lm_head(h)
        loss = nn.functional.cross_entropy(logits[:, :-1].flatten(0,1), labels[:, 1:].flatten())

        # MTP losses
        prev_h = h
        for k, mod in enumerate(self.mtp_modules, start=2):
            # Teacher-forcing: feed ground-truth embedding of position t+k-1
            prev_emb = self.embed(labels[:, k-1:-1])  # shifted
            mtp_h = mod(prev_h[:, :-(k-1)], prev_emb[:, :prev_h.size(1)-(k-1)])
            mtp_logits = self.lm_head(mtp_h)
            mtp_target = labels[:, k:k+mtp_h.size(1)]
            loss = loss + self.lam * nn.functional.cross_entropy(
                mtp_logits.flatten(0,1), mtp_target.flatten())
            prev_h = mtp_h

        return loss

4. 推理加速：MTP → Speculative Decoding

MTP head 天然是 speculative decoding 的 draft model — 无需额外小模型，自带免费 drafter
