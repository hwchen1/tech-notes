1. 核心理念：三合一更新

# Gated DeltaNet 融合三种机制:
# 1. Gate (from GLA) — 选择性遗忘
# 2. Delta (from DeltaNet) — 精确覆盖
# 3. Write (standard) — 新信息写入

def gated_deltanet_recurrent(q, k, v, alpha, beta):
    """
    alpha: forget gate ∈ (0,1) — 全局遗忘
    beta:  erase strength ∈ [0,1] — 精确擦除
    """
    B, H, N, d = q.shape
    S = torch.zeros(B, H, d, d)
    outputs = []

    for t in range(N):
        k_t = k[:,:,t]
        v_t = v[:,:,t]
        q_t = q[:,:,t]
        a_t = alpha[:,:,t]  # gate
        b_t = beta[:,:,t]   # erase

        # 三步更新:
        # 1. 全局衰减 (forget)
        S = a_t.unsqueeze(-1) * S

        # 2. 精确擦除 (erase old value)
        old_v = (S @ k_t.unsqueeze(-1)).squeeze(-1)
        S = S - b_t.unsqueeze(-1) \
            * k_t.unsqueeze(-1) * old_v.unsqueeze(-2)

        # 3. 写入新值
        S = S + k_t.unsqueeze(-1) * v_t.unsqueeze(-2)

        # 读取
        o_t = (q_t.unsqueeze(-2) @ S).squeeze(-2)
        outputs.append(o_t)

    return torch.stack(outputs, dim=2)

为什么需要两种遗忘？

GLA 的 gate 和 DeltaNet 的 erase 解决不同问题：

机制作用粒度
α (gate)全局衰减噪声整个状态矩阵
β (erase)精确删除旧值特定 key 位置
$S_t = \underbrace{\alpha_t \cdot S_{t-1}}_{\text{全局衰减}} - \underbrace{\beta_t \cdot k_t (S \cdot k_t)^\top}_{\text{精确擦除}} + \underbrace{k_t v_t^\top}_{\text{写入}}$
互补性：Gate 处理"缓慢累积的噪声"（没有明确 key 的垃圾信息），Erase 处理"过时但有明确 key 的旧信息"。两者结合 = 最完善的记忆管理。

表达力最强的 A_t

Gated DeltaNet 的 A_t = α·I - β·kk^T 是当前所有 linear attention 中表达力最强的状态转移矩阵：

$A_t = \alpha_t I - \beta_t k_t k_t^\top$ $\text{包含 GLA 和 DeltaNet 为特例}$

它可以通过不同的 α, β 值自动退化为任何前序方法。模型学到的 α, β 分布显示两种机制都被积极使用。
