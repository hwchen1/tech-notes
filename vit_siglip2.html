2. NaFlex: 可变分辨率训练

# NaFlex: Native Flexible Resolution
# 核心思想: 固定 token 数量, 不固定分辨率

class NaFlexPatchEmbed(nn.Module):
    def __init__(self, patch_size=16, max_tokens=729):
        self.patch_size = patch_size
        self.max_tokens = max_tokens  # 27×27

    def forward(self, image):
        # 1. 保持原始宽高比
        # 2. Resize 使得 patch 总数 ≤ max_tokens
        H, W = image.shape[-2:]
        aspect = W / H

        # 找到最优 grid: h_patches × w_patches ≤ 729
        # 且 w_patches/h_patches ≈ aspect ratio
        h_p = int((self.max_tokens / aspect) ** 0.5)
        w_p = self.max_tokens // h_p

        # Resize to exact patch grid
        target_h = h_p * self.patch_size
        target_w = w_p * self.patch_size
        image = resize(image, (target_h, target_w))

        # Patch embed (标准)
        patches = patchify(image, self.patch_size)
        return patches  # [B, h_p*w_p, D]

# 关键: 2D RoPE 位置编码 (非固定 PE!)
# 2D-RoPE 根据实际 grid 大小编码位置
# → 自然适应任意 aspect ratio

为什么 NaFlex 重要

传统 ViT 强制 resize 到正方形会导致两个问题：

问题影响
宽高比变形物体比例失真
随机裁剪关键内容可能被裁掉
固定分辨率小物体细节丢失
$\text{NaFlex}: h_p \times w_p \leq N_{\max}$ $\text{s.t. } \frac{w_p}{h_p} \approx \frac{W_{\text{orig}}}{H_{\text{orig}}}$

NaFlex 的约束：固定总 token 数（如 729 = 27×27），但 grid 形状自适应原图宽高比。配合 2D RoPE 位置编码，模型天然泛化到任意分辨率。

实际效果：NaFlex 在 OCR、文档理解、细粒度识别等需要保持原始空间信息的任务上提升显著 (OCRVQA +3.2%, TextVQA +2.1%)。这也是为什么 Gemma4-VL 选择 SigLIP2 作为 vision encoder。

3. 多目标训练与自蒸馏细节

# SigLIP2 Self-Distillation 详细实现:

# Step 1: 先训练最大 Teacher 模型
teacher = SigLIP2_g16(params='1.1B')
teacher.train(data=WebLI, loss='sigmoid_contrastive')
teacher.freeze()  # 训好后冻结

# Step 2: Student 训练时加入蒸馏 loss
class SigLIP2Distillation(nn.Module):
    def __init__(self, student, teacher):
        self.student = student   # e.g. B/16, L/16
        self.teacher = teacher   # g/16 (frozen)
        # 投影头: student dim → teacher dim
        self.proj = nn.Linear(
            student.d_model, teacher.d_model
        )

    def distill_loss(self, images):
        # Student forward
        s_feat = self.student.encode_image(images)
        s_cls = s_feat[:, 0]  # CLS token
        s_proj = self.proj(s_cls)  # 投影到 teacher 维度

        # Teacher forward (no grad)
        with torch.no_grad():
            t_feat = self.teacher.encode_image(images)
            t_cls = t_feat[:, 0]  # CLS token

        # Feature-level MSE (不是 logit match!)
        # 直接对齐 representation space
        loss = F.mse_loss(s_proj, t_cls)
        return loss

    def total_loss(self, images, texts):
        L_sig = sigmoid_contrastive(images, texts)
        L_cap = captioning_loss(images, texts)
        L_dis = self.distill_loss(images)
        L_mim = masked_prediction_loss(images)
        # 联合优化四个 loss
        return L_sig + 0.5 * L_cap + 1.0 * L_dis + 0.5 * L_mim

# 关键设计决策:
# 1. Feature-level (不是 logit/softmax distill)
# 2. CLS token 对齐 (全局 representation)
# 3. 线性投影头处理维度不匹配
# 4. Teacher 完全 frozen (不更新)
# 5. 同一张图 (无 augmentation 差异)

# === Masked Prediction (第5个目标) ===
# 类似 MAE/BEiT 的自监督目标
def masked_prediction_loss(images):
    # 1. 随机遮盖 ~50% patch tokens
    patches = patchify(images)
    mask = random_mask(ratio=0.5)
    visible = patches[~mask]

    # 2. Encoder 只看 visible patches
    encoded = encoder(visible)

    # 3. 用轻量 decoder 预测被遮盖 patches
    # 预测目标: teacher 的 patch features
    pred = mask_decoder(encoded, mask_positions)
    with torch.no_grad():
        target = teacher.get_patch_features(images)
        target = target[mask]

    # 4. MSE on masked positions
    return F.mse_loss(pred, target)

# 作用: 强化局部空间推理能力
# 对 dense 任务 (分割/检测/OCR) 贡献大

Self-Distillation + Masked Prediction

SigLIP2 的蒸馏不是传统的 logit matching（对齐输出概率），而是 Feature-level Distillation：

$\mathcal{L}_{\text{distill}} = \| \text{proj}(f_s(x)_{\text{CLS}}) - f_t(x)_{\text{CLS}} \|^2$ $\mathcal{L}_{\text{mask}} = \| \text{dec}(f_s(x_{\text{vis}})) - f_t(x)_{\text{masked}} \|^2$
训练目标对齐层级贡献
Sigmoid ContrastiveGlobal (image↔text)语义对齐
Self-DistillationGlobal (CLS→CLS)小模型质量↑
Masked PredictionLocal (patch-level)dense/空间理解↑
CaptioningSequential (token生成)细粒度语义↑

Masked Prediction 的三个特殊之处：

① 预测 teacher feature 而非 raw pixels — 跟 MAE 重建像素不同，SigLIP2 遮盖 ~50% patches 后让 student 预测 teacher 对应 patch 位置的特征向量，本质上是 patch-level distillation。

② 与 Self-Distillation 形成 global+local 互补 — Self-Distillation 做 CLS-level 全局对齐，Masked Prediction 做 patch-level 局部对齐，两者共同覆盖全局语义和局部空间信息。

③ 不需要 heavy decoder — 因为目标在 feature space 而非 pixel space，无需 MAE 那样的多层 decoder，直接用轻量 linear projection 对齐维度即可，训练开销小。

五目标联合训练：SigLIP2 = Sigmoid Contrastive + Captioning + Self-Distillation (CLS) + Masked Prediction (patch) + Multi-Positive。前两者提供监督信号，后两者提供自监督/半监督信号，multi-positive 增强鲁棒性。
