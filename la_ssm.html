SSM 基础

State Space Model 从连续到离散 — 理解 Mamba/GLA/DeltaNet 的数学基础

1. 什么是 State Space Model？

# State Space Model (状态空间模型)
# 来自控制论/信号处理, 描述动态系统

# ===== 连续形式 (原始) =====
# dx/dt = A·x(t) + B·u(t)   # 状态更新
# y(t)  = C·x(t) + D·u(t)   # 观测/输出
#
# x(t) ∈ R^N — 隐状态 (N 维)
# u(t) ∈ R   — 输入信号
# y(t) ∈ R   — 输出信号
# A ∈ R^(N×N) — 状态转移矩阵
# B ∈ R^(N×1) — 输入映射
# C ∈ R^(1×N) — 输出映射

# ===== 离散化 (关键步骤!) =====
# 连续 → 离散, 用步长 Δ
# Zero-Order Hold (ZOH):
#   Ā = exp(Δ·A)
#   B̄ = (exp(Δ·A) - I)·A⁻¹·B ≈ Δ·B
#
# 离散递推:
#   x_t = Ā·x_{t-1} + B̄·u_t
#   y_t = C·x_t

# ===== 为什么叫 "State Space"? =====
# x_t 是系统的完整状态
# 知道 x_t 就能预测所有未来输出
# 不需要记住完整历史!
# → 这就是为什么 SSM 推理是 O(1) 内存

从控制论到语言模型

SSM 源自 1960 年代的控制理论（卡尔曼滤波器）。2020 年后被用于序列建模：把 token 序列视为离散时间信号，隐状态 \(x_t\) 压缩了所有历史信息。

控制理论语言模型
输入信号 utoken embedding
状态 x压缩记忆
输出 ynext token prediction
A (状态转移)记忆衰减/保留
B (输入映射)新信息写入
C (输出映射)记忆读取
SSM 的核心优势: (1) 训练时可以展开为卷积 O(N log N), (2) 推理时等价 RNN O(1) 内存。这种双重性 (duality) 是 Transformer 不具备的。

2. 从 S4 到 Mamba — 选择性 SSM

# S4 (2021): 结构化状态空间
# 问题: A 矩阵太大 (N²), 无法学习
# 解法: HiPPO 初始化 + 对角化
#   A = diag(λ₁, ..., λ_N)  (复数对角)
#   → 每个维度独立衰减/振荡

# S5 (2022): 简化为实对角
# DSS, GSS: 各种对角化变体

# Mamba (S6, 2023): 选择性 SSM
# 关键创新: 让 B, C, Δ 都依赖输入!
# 之前: A, B, C 是固定参数
# Mamba: B(x), C(x), Δ(x) 随 token 变化
#
# 效果: 模型可以"选择"记忆什么
#   Δ 大 → 更多记忆当前 token
#   Δ 小 → 更多保留历史
#   B(x) → 选择性输入门
#   C(x) → 选择性输出门

# Mamba-2 (SSD, 2024): SSM ≡ Linear Attention
# 证明: 当 A = scalar·I 时,
#   SSM 的并行形式 = 带衰减的 Linear Attention!
#   Y = (L ⊙ CB^T) · X
#   其中 L 是因果衰减 mask
# → 统一了两个研究方向!

# 这就是为什么 SSM 和 Linear Attention 总一起出现

选择性: SSM 的分水岭

S4 之前的 SSM 参数都是固定的——模型对所有 token 一视同仁。Mamba 让参数随输入变化（data-dependent），模型可以主动选择记忆/遗忘。这等价于 GLA 的门控机制。

Mamba (SSM)Linear Attention
Δ (step size)α (decay gate)
B (input map)k (key)
C (output map)q (query)
x (input)v (value)
Mamba-2 的统一定理: 把 A 限制为标量×单位阵后，SSM 和 Linear Attention 在数学上完全等价。区别只是参数化方式和初始化不同。这解释了为什么它们性能接近。

3. SSM vs Linear Attention vs RNN — 统一视角

# 三种视角看同一件事:
#
# === SSM 视角 ===
# x_t = Ā·x_{t-1} + B̄·u_t    (状态递推)
# y_t = C·x_t                  (读取)
# 来源: 控制论, 连续系统离散化
# 优势: 有离散化理论指导设计 (如 Mamba-3)
#
# === Linear Attention 视角 ===
# S_t = α_t·S_{t-1} + k_t⊗v_t  (记忆更新)
# o_t = q_t^T · S_t             (查询)
# 来源: Transformer attention 的线性化
# 优势: 可设计丰富的更新规则 (delta rule等)
#
# === RNN 视角 ===
# h_t = f(h_{t-1}, x_t)         (递推)
# y_t = g(h_t)                  (输出)
# 来源: 经典序列建模
# 优势: 概念简单, 推理 O(1)
#
# 三者关系:
# SSM (连续→离散) ⊃ Linear RNN (线性递推)
# Linear RNN ≡ Linear Attention (Mamba-2 证明)
# Nonlinear RNN ⊃ Linear RNN (M²RNN 等)

# 实践中的选择:
# 纯 SSM 参数化: Mamba 系列
# 纯 LA 参数化: GLA, DeltaNet, GDN
# 两者混合: Hymba, Super Apriel
# 加入非线性: Titans, M²RNN

为什么这么多名字？

SSM、Linear Attention、Linear RNN 经常混用，是因为 Mamba-2 证明了它们在数学上等价（当 A 为标量时）。但不同的视角带来不同的设计灵感：

SSM 视角启发了 Mamba-3 的复数态和离散化改进；Linear Attention 视角启发了 Delta Rule 系列的擦写机制；RNN 视角启发了 M²RNN 的非线性突破。

记住这个等价关系: Mamba 的 (B, C, x, Δ) = Linear Attention 的 (k, q, v, α)。参数化不同但计算相同。这是理解整个领域的关键。
