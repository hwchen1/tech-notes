MDN (Momentum DeltaNet)

Delta Rule + 动量加速 — 二阶递推系统 + 复数特征值 | Huang, Liu et al. | 2026.05 (arxiv 2605.05838)

1. 核心理念 — 从 SGD 到 Momentum

# 统一视角: Linear Attention = Online Optimization
# 状态矩阵 S 是 "fast weights"
# 每步更新 = 对 loss 做一步优化
#
# Loss: L(S) = 0.5 * ||v_t - S^T · k_t||²
#
# DeltaNet  = plain SGD (无衰减)
# GDN       = SGD + weight decay (α 衰减)
# KDA       = SGD + per-param decay (channel-wise α)
# MDN       = SGD + Momentum  ← NEW!
#
# 动量更新:
# M_t = μ_t · M_{t-1} + η_t · ∇L(S_{t-1})
# S_t = α_t · S_{t-1} - β_t · M_t
#
# 展开为二阶递推:
# S_t = (α̃_t·I - β̃_t·k_t·k_t^T)·S_{t-1}
#      - γ̃_t · S_{t-2}
#      + β̃_t · k_t · v_t^T
#
# 关键: S_t 依赖 S_{t-1} 和 S_{t-2}!
# 这是一个二阶动力系统

# 退化关系:
# μ=0, η=1, p=αk → Gated DeltaNet
# μ=0, η=1, α=1, p=k → DeltaNet
# p=0 → Mamba2 / GLA (纯衰减)

MDN 将优化中的动量概念引入 linear attention 的状态更新，减少记忆振荡

为什么动量有效？

在优化中，momentum 让梯度更新积累方向性，减少振荡。在记忆管理中，这意味着 MDN 可以更稳定地追踪变化的信息，不会因单步噪声而丢失重要模式。

SGD（DeltaNet/GDN）每步只看当前梯度，容易在狭窄山谷中来回震荡。Momentum 累积历史方向，让更新路径更平滑、收敛更快。

核心创新：MDN 是第一个在 linear attention 中实现严格逐步动量的方法。之前的 TTT/LMM 用 blockwise momentum（破坏因果性），MDN 通过几何重排序实现了因果正确的 chunkwise 并行。

退化关系

参数设置退化为
\(\mu=0, \eta=1, p=\alpha k\)Gated DeltaNet
\(\mu=0, \eta=1, \alpha=1, p=k\)DeltaNet
\(p=0\)Mamba2 / GLA

2. 二阶动力系统与稳定性

# 特征值分析:
# 一阶系统 (GDN): 特征值 ∈ R, 在 (-1,1) 内
# 二阶系统 (MDN): 特征值可以是复数共轭!
#
# 复数特征值 → 阻尼振荡行为
# = 相位感知记忆 (phase-aware memory)
#
# 稳定性约束:
# 无约束 → 负实部 → 符号翻转 → NaN!
# 解法: "象限约束" — 限制在第1/4象限
#   β_t ≤ 1 - α_t
#   μ_t ∈ (e^{-1}, 1)
#
# 实现:
#   θ = sigmoid(·)
#   α_max = cos²(θ)
#   β_max = sin²(θ)
#   保证 α_max + β_max = 1

# Chunkwise 并行算法:
# 挑战: 逐步动量天然串行
# 解法: 几何重排序
#   嵌套求和 ∑ᵢ∑ⱼ≤ᵢ → 列式扫描 ∑ᵢ∑ⱼ≥ᵢ
#   所有系数用 log-domain 前缀和计算
#   Triton kernel 实现

# 内存优化:
# 存储校正值 V_tilde (而非 S 和 M)
# backward 时重建状态

MDN 的复数特征值允许阻尼振荡，实现比单调衰减更丰富的记忆动态

复数特征值的物理意义

GDN 的记忆只能单调衰减（实特征值），但自然语言中信息的重要性是振荡的（一个 entity 可能暂时不重要但后面又变重要）。复数特征值允许阻尼振荡，类似物理中的弹簧系统。

实特征值 \(\lambda \in \mathbb{R}\)：记忆 \(\propto \lambda^t\) 单调衰减。复数特征值 \(\lambda = re^{i\theta}\)：记忆 \(\propto r^t \cos(\theta t)\) 振荡衰减。

稳定性消融

配置结果
有象限约束正常收敛
无象限约束NaN at step ~2k
理论驱动的设计：MDN 的稳定性约束是通过动力系统分析推导出精确的参数范围，而非靠经验 tuning。象限约束保证所有特征值停留在单位圆内的右半平面，避免符号翻转引发的数值爆炸。

3. 实验结果 — 全面超越 GDN

# 1.3B, 100B tokens:
#                 Wiki PPL  Retrieval  Commonsense

# GDN             18.51     32.80      57.44
# KDA             19.24     32.62      58.56

# Multi-Query: MDN +11.45 over best baseline
# Multi-Value: MDN +8.95 over best baseline

# LongBench (16K):
# MDN: 20.18 (vs GDN 19.28, KDA 18.62)

# Hybrid (7:1 linear:attn):
# Hybrid MDN > other hybrids at 3:1 ratio
# → MDN 减少对 full attention 层的依赖

# 训练吞吐: 与 Mamba2/KDA 相当
# 解码延迟: 与 GDN 几乎一致 (线性复杂度)

MDN 在 PPL 和检索任务上全面领先，尤其在需要维护多关联的场景中优势巨大

动量的最大收益: 检索任务

MDN 在常规 PPL 上提升适度 (18.51 → 18.03)，但在 multi-needle retrieval 上改进巨大 (+13.40)。这是因为动量帮助模型更稳定地维护多个 key-value 关联。

动量积累使得写入/擦除操作不再是"一锤子买卖"，而是有惯性的渐进过程，减少了频繁覆写导致的信息丢失。

优化理论的指导价值：MDN 证明了优化理论对 linear attention 设计的指导价值。动量是最自然的下一步，未来可能还有 Adam-like（自适应学习率）变体。从 SGD → Momentum → Adam 的路线图，为 linear attention 的发展提供了清晰方向。
