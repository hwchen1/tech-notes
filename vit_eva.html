核心创新

MAE 重建像素 → 低级特征。EVA 重建 CLIP 特征 → 直接学习语义级表征。

类似"蒸馏式 MIM"：用冻结的 CLIP 教师提供更高质量的重建目标。

$\mathcal{L}_{\text{EVA}} = \text{MSE}(\text{dec}(z_{\text{vis}}), f_{\text{CLIP}}(x_{\text{mask}}))$ $\text{vs MAE: } \mathcal{L}_{\text{MAE}} = \text{MSE}(\text{dec}(z_{\text{vis}}), x_{\text{mask}})$

迭代升级

EVA → EVA-CLIP → EVA-02 → EVA-02-CLIP：每代用上一代最强模型作为教师，bootstrap 式提升。

2. EVA-02 架构改进

EVA-02 引入 LLM 领域的最佳实践（SwiGLU, RoPE）到视觉编码器，统一视觉和语言的架构设计

$\text{EVA-02 FFN: SwiGLU}(x) = (\text{Linear}_1(x) \odot \text{SiLU}(\text{Linear}_g(x))) \cdot \text{Linear}_2$ $\text{EVA-02 Pos: 2D RoPE} — \text{支持任意分辨率外推}$ $\text{Sub-LN: } x + \text{LN}(\text{Attn}(\text{LN}(x))) \quad \text{(双重 LN 稳定深层训练)}$
